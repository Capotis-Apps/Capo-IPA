## 🛠️ Métodos de Instalación

Para instalar cualquiera de estos archivos `.ipa`, necesitas usar una herramienta de sideloading en tu iPhone o iPad:

* **TrollStore:** Recomendado si tu versión de iOS es compatible (no requiere firmar cada 7 días).
* **AltStore / SideStore:** Excelente opción usando tu ID de Apple (requiere PC/Mac para la configuración inicial).
* **Sideloadly:** Método rápido desde el ordenador mediante cable o Wi-Fi.
